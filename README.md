# 배열에 '\n' 이 들어가는데 설명좀요
이게 정확하진 않는데 제가 아는 피셜로는 
<pre>
int array[100]= { '\n' };
</pre>
이렇게 될 경우 101개의 자리에 모두 NULL 값이 들어가는걸로 알고 있습니다. 
<pre>
int array[100]= { 0, };
</pre>
위와 같이 진행 해도 큰 문제는 없지만 저는 뭔가 0 보다는 NULL이 들어간게 맘이 편하더라고요..
